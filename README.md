# CruxAffairs

AI-powered Current Affairs Intelligence Platform for competitive exam aspirants.

CruxAffairs automatically collects news from trusted sources, processes the content using AI, extracts exam-relevant information, and presents concise, structured summaries for faster learning and revision.

## Features

* Automated news aggregation
* AI-generated summaries
* Exam-focused fact extraction
* Subject-wise categorization
* Daily current affairs dashboard
* Searchable archives
* Revision mode
* Weekly and monthly compilations
* PDF export support
* Personalized learning experience

## How It Works

```text
News Sources
     ↓
Data Collection
     ↓
Content Cleaning
     ↓
AI Summarization
     ↓
Fact Extraction
     ↓
Category Classification
     ↓
Dashboard Publishing
```

* Frontend: Next.js / React
* Backend: Node.js
* Database: PostgreSQL(Neon)
* AI Processing: LLM-based summarization and classification
* Authentication: JWT / OAuth
* Deployment: Microsoft Azure
